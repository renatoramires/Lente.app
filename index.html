Oi, Renato. Sou o Lente: posso ouvir, traduzir e ajudar com o que estiver na câmera. Esta versão é um protótipo web; para usar Claude de verdade, conecte um backend seguro.

Câmera do celular

Use o celular como câmera para ler placas, textos, produtos ou objetos. A imagem não sai do aparelho neste protótipo.

Tradução simultânea

Modo pensado para palestra, viagem, curso ou cinema: capta áudio, transcreve e mostra a tradução. A integração final deve chamar Claude/Whisper no servidor.

Conectar óculos

Compatível como conceito com óculos Bluetooth/microfone e celular como câmera. Even G2 pode servir como display/microfone; TopStep/AIBuds depende do app/SDK do fabricante.

Backend seguro

Nunca coloque chave da Anthropic/OpenAI no HTML. Crie uma API intermediária para chamar Claude, transcrição e tradução.
